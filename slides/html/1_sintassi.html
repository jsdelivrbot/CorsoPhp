Cos'è un file php?

  • Ha estensione ".php"
  • Può contenere HTML, CSS, Javascript, ecc.
<html>
 <head>
  <title>Test PHP</title>
 </head>
 <body>
 <p>Hello World</p>
 </body>
</html><body>
<p>Hello World</p>
</body>

Come funziona

  • Il Client invia una richiesta al Web Server
  • Il Web Server direziona la richiesta verso PHP
  • PHP elabora la richiesta e produce una pagina html
  • Il Web Server restituisce la pagina all'utente

PHP: Hypertext Preprocessor

  • Nasce nel 1995
  • Multipiattaforma
  • Molte librerie
  • Supporto MySQL, ecc
