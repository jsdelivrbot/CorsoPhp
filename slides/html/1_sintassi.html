Cos'è un file php?

  • Ha estensione ".php"
  • Può contenere HTML, CSS, Javascript, ecc.
<html>
 <head>
  <title>Test PHP</title>
 </head>
 <body>
 <?php echo "Hello World!<p>"; ?>
 </body>
</html>

Come funziona

  • Il Client invia una richiesta al Web Server
  • Il Web Server direziona la richiesta verso PHP
  • PHP elabora la richiesta e produce una pagina html
  • Il Web Server restituisce la pagina all'utente

Hello World!

Il primo script


<!DOCTYPE HTML>
<html>
<head>
<meta charset="UTF-8">
<title>Hello World!</title>
</head>
<body>
<?php
    //Write out Hello World!
    echo 'Hello World!';
?>
</body>
</html>

Il comando echo è usato per stampare del testo
