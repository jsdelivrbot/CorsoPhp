Variabili

Una variabile è un contenitore di informazioni

Può contenere un numero, un testo, un valore vero/falso...

Tipo Cosa contiene Esempio
Integer Num. intero 5
Double Num. reale 10.5
String Testo "Hello"
Boolean Vero o Falso 1
Array Lista di elementi 1, 4, 3
Object Istanza di una classe Persona

Dichiarare una variabile

Dichiarare una variabile

PHP è un linguaggio debolmente tipizzato. Questo significa che non bisogna specificare il tipo di variabile che si vuole utilizzare. PHP converte la variabile al tipo di dato corretto, in base al suo valore.

Dichiarare una variabile

Una variabile inizia con $ seguito da un nome


<?php
   $x = 5;
   $y = 10.5;
   $testo = "Sono una variabile di testo";
?>

Le varibili sono case sensitive, $x è diverso da $X e il nome non può iniziare con un numero

Operazioni tra varibili

Si possono fare le normali operazioni algebriche


<?php
   $x = 11;
   $y = 5;

   $z = $x + $y; // 16
   $z = $x - $y; // 6
   $z = $x * $y; // 55
   $z = $x / $y; // 2.2
   $z = $x % $y; // 1
?>

% è il modulo ovvero il resto della divisione

Stampare una variabile

Per stampare una variabile usiamo echo


<?php
   $x = 5;
   $y = 10.5;
   $testo = "Sono una variabile di testo";

   echo $x;
   echo $y;

   echo "$x + $y = " . $x + $y;
   echo '$x + $y = ' . $x + $y;
?>
  • le stringhe si possono concatenare con .
  • ciò che è compreso tra " viene valutato
  • ciò che è compreso tra ' non viene valutato

Costanti

Una costante è un contenitore di informazioni

ma il suo valore non può essere cambiato una volta definito

Creare una costante

Creare una costante

Si usa define(nome variabile, valore)


<?php
   define("GIORNI_SETTIMANA", 7);
   define("MSG_BENVENUTO", "Benvenuto!");

   echo MSG_BENVENUTO;
   echo "In una settimana ci sono " . GIORNI_SETTIMANA;
?>

Challenge

Scrivi qui sotto uno script PHP che:
  • calcoli quante ore ci sono in un dato numero di anni
  • il numero di anni deve essere memorizzato in una variabile

<!DOCTYPE HTML>
<html>
<head>
<meta charset="UTF-8">
<title>Challenge</title>
</head>
<body>

</body>
</html>

Soluzione


<!DOCTYPE HTML>
<html>
<head>
<meta charset="UTF-8">
<title>Challenge</title>
</head>
<body>
<?php
   // Costanti
   define("ORE_GIORNO", 24);
   define("GIORNI_ANNO", 365);
   // Variabili
   $anni = 3;
   echo "In $anni anni ci sono " . $anni * ORE_GIORNO * GIORNI_ANNO . ' ore';
?>
</body>
</html>
